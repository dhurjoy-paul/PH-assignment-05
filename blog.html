Question-1: What are the different ways to select an element in the DOM?

Elements can be selected in the DOM by many ways. All are given below:

document.getElementById(id)
document.getElementsByTagName(tag)
document.getElementsByClassName(class)
document.querySelector(selector)
document.querySelectorAll(selector)

But for selecting only one element should use below these. Because all these will return only one element.

document.getElementById(id) or
document.querySelector(selector)

Question-2: What is the difference between innerHTML, innerText, and textContent ?

textContent can give all text of a selected element even if text is styled with "display: none" means it ignores CSS styles. But innerText can't do that. It prioritizes CSS styles. On the other hand, innerHTML will give full HTML markup of a selected element, including its child elements and all their tags.

Question-3: What is event delegation in the DOM?

In the DOM, event delegation means assigning an event listener for children of a parent element that event listener will assign on parent element but not in every child and for every new added child element that event listener will also work.

Question-4: What is event bubbling in the DOM?

In the Document Object Model (DOM), Event bubbling is a process, of occurring events from down to up means targeted element to its parent and its parent until it gets to the root element unless forcefully stops. Event bubbling is event's natural flow.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating an element : I will create an element by document.createElement("tagName")
Adding that element : For adding an element I will follow 3 steps:

  • 1) 1st I will store that element on a variable by const elementVar = document.createElement("tagName")
  • 2) Then I will store the parent element on a different variable by its ID const parentVar = document.getElementById("id")
  • 3.1) Then I will write this document.parentVar.appendChild(elementVar) code to add that created element on parent element as "last child"
  • 3.2) If i have to add that element as first child then I will write this parentVar.insertBefore(elementVar, parentVar.firstElementChild) code.

Removing that element : I can remove that element by using its var name. This method will remove that element: parentVar.removeChild(elementVar);
